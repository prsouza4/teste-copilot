# Usuário API

## Descrição

API para cadastro e gerenciamento de usuários. Permite realizar operações de CRUD (Create, Read, Update, Delete) para usuários.

## Funcionalidades

- Cadastro de usuários com validação de CPF e e-mail.
- Operações de CRUD completas.
- Integração com RabbitMQ para publicação de eventos.
- Logs estruturados com Serilog.
- Suporte a CQRS e Result Pattern.
- Tratamento global de exceções.

## Tecnologias Utilizadas

- .NET 10
- ASP.NET Core
- Entity Framework Core
- SQL Server
- RabbitMQ
- Serilog
- FluentValidation
- MediatR
- Aspire para desenvolvimento local
