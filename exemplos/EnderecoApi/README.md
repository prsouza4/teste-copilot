# EnderecoApi

## Descrição

Uma API para gerenciar endereços, permitindo criar, listar, atualizar e excluir endereços. A API também impede o cadastro de endereços duplicados.

## Requisitos

- .NET 8 SDK
- SQL Server (ou LocalDB para desenvolvimento)
